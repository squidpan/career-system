# Resume Bullet Ranking Standard v0.4.6.1

## Purpose

Rank existing resume bullets according to resume-tailoring signals.

## Inputs

- Resume tailoring report
- Teal-export resume markdown

## Output

- Top resume bullets to promote
- Bottom resume bullets to de-emphasize
- Ranking rationale via matched terms

## Design

v0.4.6.1 ranks existing bullets. It does not rewrite bullets or generate final resumes.
